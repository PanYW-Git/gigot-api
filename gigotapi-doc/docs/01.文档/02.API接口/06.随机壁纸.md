### 接口信息

- 接口状态 ： <Badge text="正常"/>
- 请求方式 ：`GET`
- 返回格式 ：`JSON`
- 扣除积分数 ：`1`

### 请求地址
```shell
http://gateway.panyuwen.top/api/randomWallpaper
```

### 请求参数

| 参数名 | 必选 | 类型 |                   描述                    |
|:---:|:---:|:---:|:---------------------------------------:|
|   method   |  否  |  string  |         输出壁纸端mobile、pc、zsy默认为pc         |
|   lx   |  否  |  string  | 输出分类meizi、dongman、fengjing、suiji，为空随机输出 |

### 响应参数

|  参数名称   |  类型  |   描述   |
|:----------:|:------:|:---------:|
|    code     |  int   |  响应码  |
| data.imgurl | string | 图片地址 |
|   message   | string | 响应描述 |
