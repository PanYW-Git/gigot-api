- **JDK 8+**
- **Maven or Gradle**

## SpringBoot

### 🍊Maven
在项目的pom.xml的dependencies中加入以下内容:
```xml
<dependency>
    <groupId>top.panyuwen/groupId>
    <artifactId>gigoapi-sdk</artifactId>
    <version>0.0.4</version>
</dependency>
```
### 🍐Gradle
```
implementation group: 'top.panyuwen', name: 'gigoapi-sdk', version: '0.0.4'
```
